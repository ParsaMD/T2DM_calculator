## Contents ##

- `streamlit_app.py`  
    The Streamlit web application script for inputting clinical parameters and obtaining risk predictions.

- `std_scaler_7param.pkl`  
    Pre-trained StandardScaler for data standardization.

- `calibrated_svm_7param.pkl`  
    Calibrated SVM model for risk prediction.

- `calibrated_stacking_7param.pkl`  
    Calibrated Stacking ensemble model for risk prediction.

- `requirements.txt`  
    Required Python packages to run the app.
